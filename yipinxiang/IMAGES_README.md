## 图片规格建议

### 轮播图
- 尺寸：750×400px（2:1比例）
- 格式：PNG或JPG
- 风格：温馨、甜美、有食欲感

### 产品图片
- 尺寸：建议1:1正方形，最小300×300px
- 格式：PNG（透明背景）或JPG
- 要求：清晰展示产品细节，背景干净

### 图标
- 尺寸：建议60×60px
- 格式：PNG（透明背景）
- 风格：简洁、可爱、与主题色协调

### 头像
- 尺寸：建议120×120px
- 格式：PNG或JPG
- 风格：真实或卡通风格

## 使用建议
1. 所有图片建议使用WebP格式以减小文件大小
2. 产品图片建议使用白色或浅色背景
3. 图标建议使用矢量图标库生成
4. 轮播图建议包含促销信息和产品亮点

## 替代方案
如果暂时没有实际图片，可以使用以下方法：
1. 使用占位图生成器（如https://placeholder.com）
2. 使用免费图库（如Unsplash、Pexels）寻找类似图片
3. 使用AI生成图片工具
4. 使用简单的色块或图标代替

## 图片优化
1. 压缩所有图片以减少加载时间
2. 使用合适的图片格式（照片用JPG，图标用PNG）
3. 实现懒加载提升性能
4. 使用CDN加速图片加载
