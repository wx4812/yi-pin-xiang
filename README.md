# 一品香蛋糕房微信小程序

## 项目概述
这是一个完整的蛋糕房微信小程序，名为"一品香蛋糕房"，包含完整的电商功能模块，界面风格温馨精美，适合蛋糕甜品店使用。

## 功能特性
- ✅ 首页：产品展示、轮播图、店铺介绍、分类导航
- ✅ 产品详情页：规格选择、数量调整、用户评价
- ✅ 购物车：商品管理、数量调整、全选/反选
- ✅ 订单系统：订单列表、状态筛选、订单详情
- ✅ 结算流程：地址选择、订单确认、提交订单
- ✅ 关于我们：店铺介绍、联系方式、小程序码分享
- ✅ 数据持久化：购物车和订单数据本地存储

## 页面结构
```
yipinxiang/
├── app.js              # 全局逻辑
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
├── sitemap.json        # 搜索索引
├── images/             # 图片资源
│   ├── tabbar/         # 底部导航图标
│   ├── icons/          # 分类图标
│   ├── products/       # 商品图片
│   └── avatars/        # 用户头像
└── pages/              # 页面目录
    ├── index/          # 首页
    ├── product/        # 产品详情页
    ├── cart/           # 购物车
    ├── order/          # 订单相关
    │   ├── order/      # 订单列表
    │   └── checkout/   # 结算页面
    └── about/          # 关于我们
```

## 技术栈
- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 本地存储（wx.setStorageSync/wx.getStorageSync）
- 组件化开发

## 界面特色
- 温馨的粉色系配色方案
- 流畅的交互动画和过渡效果
- 响应式布局适配各种屏幕
- 精美的卡片式设计
- 清晰的信息层级结构

## 使用说明
1. 使用微信开发者工具打开项目
2. 在 `app.json` 中配置小程序信息
3. 准备图片资源并放置在对应目录
4. 编译运行即可体验完整功能

## 图片资源说明
项目需要以下图片资源：
- 轮播图：banner1.png, banner2.png, banner3.png
- 店铺logo：shop-logo.png

- 产品图片：strawberry-mousse.png, mango-layer.png 等

- 底部导航图标：home.png, cart.png, order.png, user.png 及对应的激活状态

## 注意事项
1. 所有图片路径已在代码中预设，请按路径放置图片
2. 实际使用时需要申请微信小程序账号并配置相关权限
3. 支付功能需要接入微信支付API
4. 地址选择功能需要接入微信地址选择组件
